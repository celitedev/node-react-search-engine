## Requirements
 - Node 6.0.0
 - NPM 3.8.6
 - Babel 5

## Local development
 - `npm install`
 - `bower install`
 -  install RethinkDB // https://www.rethinkdb.com
 - `bash make-certs.sh 'localhost:7000'`
 - `npm start`
 - `npm run db`
 - `open https://localhost:7000`

 ## Production
 - `npm run build`
