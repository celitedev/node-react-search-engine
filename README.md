## Requirements
 - Node 6.0.0
 - NPM 3.8.6
 - Babel 5

## Local development
 - `npm install horizon -g`
 - `npm install`
 - `bower install`
 -  install RethinkDB // https://www.rethinkdb.com
 - `npm run db`
 - `hz create-cert`
 - `npm run set-schema`
 - `npm start`
 - `open https://localhost:7000`

 ## Production
 - `npm run build`
 - `npm run production`
